# Regenerating Run Configurations

## IntelliJ IDEA

Delete the run configurations and refresh Gradle. (You can do this by the run configs prompt) If the run configurations do not show up after refreshing Gradle, try closing the IDEA project and reopening it, or choosing the `.idea` directory in Project Files and refreshing it.

## Visual Studio Code

Delete `.vscode` folder and refresh Gradle.
